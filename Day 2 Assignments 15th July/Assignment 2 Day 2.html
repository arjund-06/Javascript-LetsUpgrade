Different method of strings and arrays:

There are a few methods which can be used on strings and arrays,

--> For Strings...

1. split() - This splits the string into an array of substrings.
2. toLowerCase() - This converts all the characters in the string to lowercase letters.
3. toUpperCase() - This converts all the characters in the string to uppercase letters.

--> For Arrays...

1. push() - This adds the element mentioned to the end of the array.
2. pop() - This removes the last element of the array

--> But there are a few methods which can be used both on strings and arrays...

1. indexOf() - This returns the index where the entered value is present in the string/array. 
2. lastIndexOf() - This is returns the index of the last occurence of the entered value present in the string/array.
