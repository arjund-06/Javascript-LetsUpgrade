Data types in JS

There are a few data types in Java Script,
1. Number - All numeric values.
2. BigInt - Numbers with arbitary length greater then 251 or -(251) for negative side.
3. String - All characters, must be put inside double quotes ("").
4. Boolean - True or False values.
5. Null value - It is a value which does not belong to any other data type
6. Undefined value - It is a variable for which the value isnt assigned
